# Internship Project

This project was created as part of the **Internship course** to learn and practice backend development using **Java** and **Spring Boot**.

## Technologies

The project is built using the following technologies:

- **Java 17**
- **Spring Boot**
- **Apache Maven**
- **REST API**

## Project Purpose

The main goal of this project is to gain practical experience with:

- Spring Boot application structure
- REST controller development
- Dependency management with Maven
- Running and managing a Java backend application

## Running the Project
Make sure **Java 17** and **Maven** are installed.

Run the application using:

```bash
mvn spring-boot:run
```
Or run the main class: InternshipApplication.java

The server will start on: http://localhost:8080
